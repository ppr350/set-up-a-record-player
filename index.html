SET UP A RECORD PLAYER

步驟一

確保先把唱臂（tonearm）鎖著固定，然後把唱頭蓋（headshell）插進再鎖緊，避免碰到唱針（stylus）。

步驟二

小心拿開唱臂的固定安全鎖，一隻手必須固定著唱臂以防它飄開，因為平衡這時還沒設定好。現在把唱臂拿到轉盤位子（轉盤這時不能有唱片），一隻手輔助唱臂避免它打到轉盤，另一隻手調整平衡重（counter weight）來調出一個水平對稱點讓唱臂能夠平衡地飄浮。

步驟三

現在再把唱臂放回本來的位置。要確定唱臂的升降控制棒（tonearm lifter）是在”降”模式，然後把唱臂的安全鎖鎖回去把唱臂固定。

步驟四

這個步驟是把調好了的平衡點的讀點歸零。平衡重的前半段是可以手動獨立轉動的而不影響調好了的平衡重（前提是手動轉時沒有無意的也把平衡重微轉）。小心的把讀點歸0。現在唱臂在完全平衡的狀態而且讀點是0。如果要確認是否還平衡的話（因為有不小心微調平衡重的可能行）可以重複步驟二。

步驟五

現在是要把唱針的針壓調到官方推介的範圍。這款唱針的針壓範圍是1.5g到2.5g之間。我們就把針壓調去2g吧。先確定唱臂被固定著，然後把平衡重轉到2（隨著轉動，平衡重會慢慢往唱臂中間方向移去）。這個步驟是設定唱針讀黑膠唱片的力度，太重（超過官方建議）會慢慢損壞唱片，太輕（低過官方建議）會容易跳針。

步驟六

最後我們需要設定一個抗側滑力（anti-skating）。當播放黑膠唱片時唱針會慣性自然地被吸力推去唱盤的中軸。這一步的目標是用抗側滑力來施加反作用來對抗慣性。通常我們會根據針壓的設定來決定抗側滑力的數目。先確定唱臂被固定著，然後把抗側滑力調到2。

步驟七

播放黑膠唱片。把唱片放入唱機後接下來按播放。現在先把唱臂固定鬆開，然後用升降控制棒把唱臂提升，把唱臂移去黑膠唱片的盤頭沒有溝槽（grooves）的平面，慢慢用升降控制棒把唱臂下降。

步驟八

這個階段可以是已經把黑膠唱機設定完成。只是還需要細心觀察聆聽看看播放有沒有問題，例如有沒有容易跳針，或者會不會聽到一些雜音（例如S音會破音）之類的。

最後，記得完畢播放後先用升降控制棒把唱臂提升後才按停止播放。把唱臂移回原位後，先先用升降控制棒把唱臂下降，最後才把唱臂固定（如果還要播放另一面或其他唱片就無需固定）。

保養

播放黑膠唱片前先掃掉唱片上的灰塵。按播放，然後用掃塵專用的器具從唱片內側開始掃出。

唱針也必須常清潔。只能從內到外清潔，像影片裡一樣。

拿唱片時，手不要碰唱片的溝槽，以免把手油轉移到唱片。
